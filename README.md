# ⛺ Campsite Commander App# to help users organise and manage camping gear and food supplies efficiently.

Users are able to:
- Add camping items
- Categorize items (e.g., Shelter, Food, Safety)
- Enter quantities
- Add notes or comments
- View a full checklist of all items

The app also calculates the **total number of items packed** using a loop.

---

## ✅ Features  

- ✅ Splash Screen (auto transition after 3 seconds)  
- ✅ Main Screen for user input  
- ✅ Add Gear functionality  
- ✅ Calculates total items using a loop  
- ✅ Detail screen displaying full checklist  
- ✅ Uses **parallel arrays** to store data  
- ✅ Navigation between screens (Intent)  
- ✅ Error handling for invalid input  
- ✅ Back button navigation  

---

## 🧠 How the App Works  

1. User enters:
   - Item name  
   - Category  
   - Quantity  
   - Comment  

2. When **Add Gear** is clicked:
   - Data is stored in arrays  
   - Index increases  
   - Loop calculates total quantity  

3. When **View List** is clicked:
   - Data is passed to Detail Screen  
   - Loop displays all items  

4. Detail screen shows:
   - Item  
   - Category  
   - Quantity  
   - Comments
